# 📋 Task Management

A full-stack web application developed using **Flask** and **SQLite** to help users efficiently manage their daily tasks with authentication, progress tracking, and scheduling features.

## 🚀 Features

* 🔐 User Authentication (Login, Signup, Logout)
* 🔑 Forgot Password functionality
* ➕ Add, Edit, and Delete Tasks
* 🔍 Search Tasks by Name
* 🎯 Filter Tasks by Priority and Status
* 📅 Calendar View
* 📊 Dashboard Statistics
* 📈 Progress Tracking
* ✔️ Mark Tasks as Complete
* ⏰ Automatic Status Updates (Completed/Missed)
* 📤 Export Tasks to CSV
* 👋 Personalized Welcome Message

## ⚙️ Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd Task-Scheduler
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python app.py
```

4. Open in browser:

```text
http://127.0.0.1:5000
```
